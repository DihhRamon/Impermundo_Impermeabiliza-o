05 / ANIMAÇÕES

Efeitos de Transição & Scroll

Especificações das animações do design system. Inclui a revelação de elementos com a rolagem de tela e efeitos de preenchimento e distorção de texto.

1. Revelação Gradual (Scroll Reveal)

Utiliza a API nativa do navegador IntersectionObserver para monitorar o scroll do usuário e ativar a visibilidade dos cards sem prejudicar a performance geral do site.

  • .reveal: Sobe o elemento verticalmente.
  • .reveal-left: Desloca o elemento a partir da esquerda.
  • .stagger-up: Revela sub-elementos com delays ordenados de 120ms.
.reveal
.reveal-left
.stagger-up

2. Divisão de Palavras (Split Reveal)

Divide os blocos de texto ou títulos em palavras agrupadas por contêineres CSS com overflow: hidden. Ao rolar, as palavras sobem com efeito de slide.

MÚSICA CONGELADA
